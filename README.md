# .github | AMPAV Organization Hub

Welcome to the **AMPAV (Augmented Metadata Pipelines for Audio/Video)** community health and documentation repository. This repository serves as the central hub for organization-level configurations, project standards, and the primary knowledge base.

---

## 📖 Knowledge Base & Wiki
We maintain our technical specifications, architectural decisions, and development workflows in the Wiki to ensure they are easily searchable and up-to-date.

### 👉 [Visit the AMPAV Wiki](https://github.com/AMPAV/.github/wiki)

**Key Wiki Sections:**
* **Onboarding**: Setting up your local environment and accessing IU RED.
* **Persistence Strategy**: Why we use YAML for internal persistence and how converters bridge the gap to external systems.
* **MGM Integration**: Guidelines for adding new Machine Generated Metadata tools (like Whisper or SpaCy) to the pipeline.

---

## 🏗 Project Repositories
* [`ampav-core`](https://github.com/AMPAV/ampav-core): The primary Python framework for orchestrating augmented pipelines.
* [`ampav-converters`](https://github.com/AMPAV/ampav-converters): Utility library for transforming internal YAML states into production formats (JSON-LD, Avalon).
